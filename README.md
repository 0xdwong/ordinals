# 简介

比特币序数理论(Ordinals Theory)，下面用 Ordinals 简称&#x20;

官网：https://docs.ordinals.com/&#x20;

### Ordinals 是什么&#x20;

Ordinals 由比特币核心开发者Casey Rodarmor 在2022 年2 月2 日提出BIP，2023 年1 月底上线。&#x20;

> 序数理论是一种协议，用于将序列号分配给聪（satoshis, 比特币的最小细分单位），并在交易花费时踪这些聪
